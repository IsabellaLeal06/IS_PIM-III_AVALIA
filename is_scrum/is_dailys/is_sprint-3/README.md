# 📅 IS_Dailys – Sprint 3

Esta pasta contém os registros das reuniões diárias (Daily Scrum) realizadas durante a Sprint 3 do projeto Avalia+.

---

## 🎯 Objetivo da Sprint

Avançar na modelagem backend em C#, organização do banco de dados, refinamento das interfaces e evolução estrutural do sistema Avalia+.

---

## 📋 Atividades da Sprint

Durante esta Sprint foram realizadas atividades relacionadas a:

- Desenvolvimento inicial da modelagem backend em C#  
- Estruturação do diagrama de classes  
- Organização do banco de dados  
- Desenvolvimento das primeiras interfaces e protótipos  
- Refinamento dos fluxos de navegação  
- Continuação da documentação técnica  
- Organização das Sprints e Dailys  

---

## 📌 Sobre as Dailys

As reuniões diárias tiveram como foco:

- Acompanhamento da modelagem backend  
- Organização das atividades do banco de dados  
- Revisão dos protótipos e interfaces  
- Refinamento da arquitetura do sistema  
- Alinhamento das entregas da Sprint  

As Dailys do projeto Avalia+ foram realizadas via Discord.
